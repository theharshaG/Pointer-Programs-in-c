# Pointer-Programs-in-c

C Pointer Programs

## Overview

This repository contains C programs demonstrating the concepts of pointers and their applications. These programs help in understanding memory addresses, pointer variables, dereferencing, pointer arithmetic, and using pointers with arrays.

The programs are useful for:

- Beginners learning C programming
- College laboratory exercises
- Coding interview preparation
- Logic-building practice
- Understanding memory concepts
- Embedded systems programming fundamentals

---

## Programs Included

### 1. Basic Pointer Demonstration

This program demonstrates pointer declaration, storing addresses, and accessing values using pointers.

#### Output Example

```text
Value of x = 10
Address of x = 6422292
Pointer value = 6422292
Value using pointer = 10
```

#### Concepts Used

- Pointer declaration
- Address operator (&)
- Dereferencing operator (*)
- Memory access

---

### 2. Modify Variable Using Pointer

This program modifies the value of a variable using a pointer.

#### Output Example

```text
x = 50
```

#### Concepts Used

- Pointer dereferencing
- Updating variable values
- Memory manipulation

---

### 3. Sum and Largest Number Using Pointers

This program accepts two numbers and calculates the sum and largest value using pointers.

#### Output Example

```text
Enter first number:
10

Enter second number:
20

Sum = 30
Largest = 20
```

#### Concepts Used

- Pointer variables
- User input
- Arithmetic using pointers
- Conditional statements

---

### 4. Pointer Arithmetic with Array

This program demonstrates pointer arithmetic by moving a pointer to the next memory location.

#### Output Example

```text
Address = 6422292
New Address = 6422296
Value = 20
```

#### Concepts Used

- Arrays and pointers
- Pointer increment
- Memory addresses

---

### 5. Array Traversal Using Pointers

This program traverses an array using pointers without array indexing.

#### Output Example

```text
10 20 30 40 50
```

#### Concepts Used

- Arrays and pointers
- Pointer traversal
- Iteration using pointers
